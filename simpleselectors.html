Simple selectors in CSS 




 Simple selectors selects elements based on tag name, id and class 


    	
	
  1.  Element selector 
    2. 
	
    • The CSS element selector selects HTML based on the element name 
    

	
  2.  Id selector 
    3. 
	
      
	
    • The CSS id selector uses the id attribute of an HTML element to select a specific element
      • 
	
    • The id of an element is unique within a page, so the id selector is used to select one unique element
      • 
	
    • To select an element with a specific id, write a hash(#) character followed by the id name of the element
      • 
	
    • Note : An id name cannot start with a number
      • 
	
    
	
	
  3.  Class selector 
    4. 
	
      
	
    • The class selector selects HTML element with specific class attribute
      • 
	
    • To select elements with a specific class, write a period(.) character followed by the class name of the element
      • 
	
    • Note : A class name cannot start with a number
      • 






	
Universal selector

	
The universal selector selects all HTML elements on the page using asterisk(*)
