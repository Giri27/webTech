CSS and its types

CSS Cascading style sheet is the language used to style HTML document. It describes how HTML elements are to be displayed on screen. Css saves lot of work and can controls layout of multiple pages at once. Css rule consists of selector and a declaration block. The selector contains HTML element you want to style. The declaration contains one or more declaration separated by semicolons. Each declaration contains a css property name and value separated by colon. Multiple declarations are separated with semicolons and surrounded by curly braces.

There are three ways of CSS for inserting a style sheet.

  1. External CSS
  2. Internal CSS
  3. Inline CSS

External style sheets are defined within the 'link' element inside the 'head' element of an HTML file. With an external style sheets you can change the look of an entire website by changing one file. External CSS files can be written in any text editor and saved with '.css' extension. The external '.css' does not conatin any HMTL tags.

Internal style sheet is used if one single page is to be styled. The internal style is defined inside the style tag in the head section.

Inline style is used to style single element. To use inline CSS add the style attribute to the relevant element, it can contain any CSS property.

Inline has the highest priority and will override external, internal and browser defaults.
